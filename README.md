# Liamwh Raff

## How do I install these formulae?

`brew install liamwh/raff/<formula>`

Or `brew tap liamwh/raff` and then `brew install <formula>`.

Or, in a `brew bundle` `Brewfile`:

```ruby
tap "liamwh/raff"
brew "<formula>"
```

## Documentation

`brew help`, `man brew` or check [Homebrew's documentation](https://docs.brew.sh).
